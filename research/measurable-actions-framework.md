# SIO-V5 Measurable Actions Framework

## Core Principle
**Every action must have a measurable outcome that can be tracked post-implementation.**

## Scoring Methodology

### Score Bands

| Band | Score Range | Description | Action Priority |
|------|-------------|-------------|-----------------|
| Dominant | 90-100 | Category leader, best-in-class | Maintain + optimize |
| Strong | 70-89 | Clear differentiation, minor gaps | Address top 3 gaps |
| Blended | 50-69 | Understandable but not distinctive | Address top 5 gaps |
| Weak | 30-49 | Significant problems, unclear | Full overhaul needed |
| Ghost | 0-29 | Invisible, critical issues | Emergency rebuild |

### Priority Scoring (0-100)

Priority is calculated based on:
- **Impact on Conversion** (40%): How much will this improve conversion rate?
- **Impact on Visibility** (30%): How much will this improve AI/search visibility?
- **Implementation Effort** (20%): Lower effort = higher priority (quick wins first)
- **Competitive Gap** (10%): How far behind competitors are you on this?

**Formula:**
```
Priority = (ConversionImpact × 0.4) + (VisibilityImpact × 0.3) + ((100 - Effort) × 0.2) + (CompetitiveGap × 0.1)
```

### Measurable Outcome Requirements

Every action must specify:
1. **Metric**: What will be measured (e.g., "5-second comprehension rate")
2. **Baseline**: Current performance (e.g., "currently 40%")
3. **Target**: Expected improvement (e.g., "expected 85%+")
4. **Timeframe**: When to measure (e.g., "within 30 days of implementation")

## Tracking Implementation

### Pre-Implementation
- [ ] Baseline metrics documented
- [ ] Success criteria defined
- [ ] Measurement method selected
- [ ] Target outcome specified

### Post-Implementation (30 days)
- [ ] Metrics re-measured
- [ ] Actual vs target compared
- [ ] Learnings documented
- [ ] Next iteration planned

### Success Thresholds

| Action Priority | Minimum Success Threshold |
|-----------------|---------------------------|
| 90-100 (Critical) | 80%+ of target achieved |
| 75-89 (High) | 70%+ of target achieved |
| 50-74 (Medium) | 60%+ of target achieved |
| <50 (Low) | 50%+ of target achieved |
